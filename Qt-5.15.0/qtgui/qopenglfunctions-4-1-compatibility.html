QOpenGLFunctions_4_1_Compatibility Class

The QOpenGLFunctions_4_1_Compatibility class provides all functions for OpenGL 4.1 compatibility profile. More...

Header: #include <QOpenGLFunctions_4_1_Compatibility>
qmake: QT += gui
Since: Qt 5.1
Inherits: QAbstractOpenGLFunctions

This class was introduced in Qt 5.1.

Detailed Description

This class is a wrapper for functions from OpenGL 4.1 compatibility profile. See reference pages on opengl.org for function documentation.

See also QAbstractOpenGLFunctions.
