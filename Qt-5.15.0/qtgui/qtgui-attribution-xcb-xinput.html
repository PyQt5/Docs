XCB-XInput

XInput 2 extension for XCB (The X protocol C-language Binding) library.

Used in XCB platform plugin if xcb-xinput was not present on the system during Qt configuration or -bundled-xcb-xinput switch was used. To force using xcb-xinput from the system, pass -no-bundled-xcb-xinput.

The sources can be found in tqtc-qt5/qtbase/src/3rdparty/xcb.
