RapidJSON

A fast JSON parser/generator for C++ with both SAX/DOM style API.

Used in the Mapbox GL plugin of Qt Location.

The sources can be found in tqtc-qt5/qtlocation/src/3rdparty/mapbox-gl-native/deps/rapidjson.
