Qt SCXML QML Types

To import the QML types into your application, use the following import statement in your .qml file:

 import QtScxml 5.15

For more information, see Instantiating State Machines.

EventConnection

Connects to events sent out by state machines

InvokedServices

Provices access to the services invoked by state machines

ScxmlStateMachine

Provides an interface to the state machines created from SCXML files

StateMachineLoader

Dynamically loads an SCXML document and instantiates the state machine
