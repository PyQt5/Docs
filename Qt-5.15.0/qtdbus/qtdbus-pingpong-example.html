D-Bus Ping Pong Example

Demonstrates a simple message system using D-Bus.

Ping Pong is a command-line example that demonstrates the basics of Qt D-Bus. A message is sent to another application and there is a confirmation of the message.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.

Run the pong application and run the ping application with the message as the argument.

 $ ./pong &
 $ ./ping Hello
 Reply was: ping("Hello") got called
