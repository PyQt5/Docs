Detailed Description

Property Documentation

popTransition : Component

The transition used on pop operation. Unless set, the popTransition is the same as pushTransition


pushTransition : Component

The transition used on push operation.


replaceTransition : Component

The transition used on replace operation. Unless set, the replaceTransition is the same as pushTransition


Method Documentation

Transition getTransition(properties)

The base implementation of this function just looks for a property named properties.name inside itself and returns it.

See also Transitions.


void transitionFinished(properties)

Handles the completion of a transition for properties. The base implementation of this function is empty.

See also Transitions.
