All Namespaces

This is a list of the main namespaces in Qt.

QAudio

Contains enums used by the audio classes

QBluetooth

Classes and functions related to Bluetooth

QColorConstants

Contains QColor predefined constants

QDBus

Contains miscellaneous identifiers used throughout the Qt D-Bus module

QGL

Specifies miscellaneous identifiers used in the Qt OpenGL module

QKnx

Contains miscellaneous identifiers used throughout the QtKnx library

QKnx::Ets

Contains miscellaneous and ETS specific identifiers used throughout the QtKnx library

QKnx::Ets::Manufacturers

Contains KNX manufacturer-specific identifiers used throughout the QtKnx library

QKnx::NetIp

QKnx::NetIp namespace defines constants, methods, and enumerations related to KNXnet/IP communication

QKnx::NetIp::CemiServer

QKnx::NetIp::CemiServer namespace defines enumerations related to KNXnet/IP communication with a cEMI server

QLocation

Contains miscellaneous identifiers used throughout the QtLocation module

QMediaMetaData

Provides identifiers for meta-data attributes

QMqtt

Contains miscellaneous identifiers used throughout the Qt MQTT module

QMultimedia

Contains miscellaneous identifiers used throughout the Qt Multimedia library

QOpcUa

Contains miscellaneous identifiers used throughout the Qt OPC UA library

QOpcUa::NodeIds

This namespace contains enums with known node ids

QPasswordDigestor

Contains functions which you can use to generate hashes or keys

QPdf

Contains miscellaneous identifiers used throughout the QtPdf module

QQuickTest

Contains all the functions and macros related to Qt Quick Test

QScxmlExecutableContent

Contains various types used to interpret executable content in state machines

QSql

Contains miscellaneous identifiers used throughout the Qt SQL module

QSsl

Declares enums common to all SSL classes in Qt Network

QTest

Contains all the functions and declarations that are related to Qt Test

QWebSocketProtocol

Contains constants related to the WebSocket standard

Qt

Qt3DAnimation

Contains classes from the Qt3DAnimation module

Qt3DCore

Contains classes that are the foundation for Qt 3D simulation framework, as well as classes that provide the ability to render using the Qt 3D framework

Qt3DExtras

Contains classes from the Qt3DExtras module

Qt3DInput

Contains classes that enable user input

Qt3DLogic

Contains classes that enable frame synchronization

Qt3DLogic::Logic

Used to import and use the module's QML types

Qt3DRender

Contains classes that enable 2D and 3D rendering

Qt3DRender::Quick

Internal namespace to import QML types

Qt3DRender::Render

Namespace used for accessing the classes Renderer and QRenderPlugin

QtAndroid

Miscellaneous functions to aid Android development

QtCoap

Contains miscellaneous identifiers used throughout the Qt CoAP module

QtConcurrent

High-level APIs that make it possible to write multi-threaded programs without using low-level threading primitives

QtDataVisualization

Contains the classes and enumerations in the Qt Data Visualization module

QtMac

Contains miscellaneous functionality specific to the macOS and iOS operating systems

QtQml

Provides functions for producing logging messages for QML types

QtWebEngine

Helper functions for the Qt WebEngine (Qt Quick) module

QtWebView

Functions that makes it easier to set-up and use the WebView

QtWin

Contains miscellaneous Windows-specific functions
