Detailed Description

This module provides classes for developing QML applications using Qt Quick 3D.

Qt Quick 3D provides the following C++ types:

QtQuick3D C++ Types

QQuick3D

Helper class for selecting correct surface format

QQuick3DGeometry

Base class for defining custom geometry

QQuick3DObject

Base class of all 3D nodes and resources
