Detailed Description

Property Documentation

vignetteColor : vector3d

The color to use for the vignetting. Qt.vector3d(0.5, 0.5, 0.5) by default.


vignetteRadius : real

Size of the vignetting. Range is [0...5]. 0.35 by default.


vignetteStrength : real

Strength of the vignetting. Range is [0...15]. 15 by default.
