Exporting from Modo

Exporting COLLADA from Modo

  1. With the mesh object selected, triangulate the mesh by pressing Shift+T.

  2. Next, select System > Preferences from the menu bar to configure the COLLADA exporter.
  3. In the preferences dialog box, locate the Collada I/O section under File I/O. Ensure that the Save Triangles as Triangles option is checked.

  4. Close the preferences dialog box.
  5. From the menu bar, select File > Export As.
  6. Select Collada in the Save as Type drop-down list.
  7. Press Save.

Exporting FBX from Modo

  1. With the mesh object selected, triangulate the mesh by pressing Shift+T.

  2. From the menu bar, select File > Export As.
  3. Select FBX in the Save as Type drop-down list.
  4. Select Save.
